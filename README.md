# YG Uninstaller

一个高效、专业的Windows程序卸载工具，具备系统托盘、批量操作、深度清理等现代化功能。

## ✨ 核心特性

- 🎯 **系统一致性**: 与Windows控制面板"程序和功能"完全一致
- 📋 **真实程序列表**: 直接读取系统注册表，显示真实已安装程序  
- 🖼️ **程序图标显示**: 自动提取并显示每个程序的真实图标
- 🔄 **智能去重**: 自动识别并去除重复的程序条目
- 📊 **响应式界面**: 动态列宽调整，完美适应窗口大小
- 🎨 **现代化设计**: 菜单栏嵌入式搜索，无边框列表，整行选中高亮
- 🗂️ **系统托盘**: 支持最小化到系统托盘，后台运行
- ⚡ **批量操作**: 多选程序，批量卸载，全选功能
- 🧹 **深度清理**: 残留文件清理，注册表清理
- 📱 **UWP支持**: Windows Store应用检测和卸载
- 🔧 **多种卸载模式**: 标准、强制、静默、深度卸载
- ⚡ **高性能**: C++11开发，启动快速，内存占用低

## 快速开始

### 系统要求
- Python 3.6+
- Windows 7+ 
- MinGW-w64 (推荐) 或 Visual Studio 2019+

### 一键构建运行
```bash
python build.py
```

这一个命令就能完成：
- 自动检测并终止正在运行的程序
- 智能清理旧版本和编译缓存
- 编译整个项目（包括资源文件）
- 生成带图标和圆角搜索框的可执行文件
- 以管理员权限启动程序

### 其他命令
```bash
python build.py --run          # 仅运行程序
python build.py --build-only   # 仅构建项目（自动处理进程冲突）
python build.py --clean        # 手动清理构建文件
python build.py --help         # 显示帮助
```

### 🔧 构建特性
- **智能进程管理**: 自动检测并终止占用文件的进程
- **多重清理策略**: 删除失败时自动重命名备份
- **编译错误恢复**: 权限问题时自动重试编译
- **详细状态反馈**: 每个步骤都有清晰的日志输出

## 项目架构

```
yguninstaller/
├── src/                   # 源代码
│   ├── core/             # 核心模块 (日志、配置、错误处理)

│   ├── ui/               # 用户界面 (主窗口)
│   └── utils/            # 工具类 (注册表操作)
├── include/              # 头文件

├── output/               # 编译输出目录
├── build.py              # 一键构建脚本
├── README.md             # 项目说明
└── PROJECT_STATUS.md     # 开发日志
```
